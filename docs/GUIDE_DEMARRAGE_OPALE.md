# Guide de démarrage — Plateforme AIDE Op@le

Bienvenue sur la plateforme académique de capitalisation Op@le de l'académie de la Guadeloupe.

## 1. Accès

Sidebar → **Ressources** → **AIDE Op@le académique**.
URL directe : `/ressources/opale`.

## 2. Pour qui ?

- **Agents comptables** des EPLE de l'académie : consulter, contribuer, voter, signaler.
- **Modérateurs académiques** (rôle `moderateur_opale`) : approuver, rejeter, demander des modifs.
- **Rectorat** (rôle `observateur_rectoral` ou `admin`) : tableau de bord, supervision.

## 3. Premières actions

### a) Consulter
1. Aller sur **Bibliothèque de fiches**.
2. Filtrer par module Op@le (Dépense, Recette, Trésorerie, Paie...).
3. Trier par utilité ou consultations.

### b) Contribuer
1. **Soumettre une fiche** (wizard 7 étapes) :
   - Identification (titre, type, module)
   - Contexte (symptôme, contexte d'apparition)
   - Diagnostic (cause identifiée, références)
   - Procédure ordonnée (étapes numérotées + vigilances)
   - Versioning Op@le (obligatoire) + date constatation
   - Visibilité (privé / académique / national)
   - Pré-publication : auto-évaluation RGPD obligatoire
2. Le système bloque la publication si une donnée sensible (UAI, INE, IBAN, email...) est détectée.
3. Soumettre pour modération.

### c) Forum Q&R
1. Poser une question publique (titre, contexte, version Op@le).
2. Les collègues répondent ; la meilleure réponse peut être promue en fiche.

## 4. Charte non négociable

- 🟢 **Complémentaire** à l'assistance officielle (DAF A3, Pléiade, Inetum). Jamais substitut.
- 🔵 Aucune donnée nominative dans les fiches ou captures.
- 🟣 Pas de reproduction des manuels Op@le officiels.
- 🟠 La version Op@le est obligatoire sur chaque fiche.
- 🔴 Les fiches « académique » et « national » sont relues avant publication.

## 5. Gamification

| Badge | Critère |
|---|---|
| Premier pas | 1 fiche publiée |
| Contributeur actif | 5 fiches publiées |
| Rédacteur d'or | 15 fiches avec utilité ≥ 70% |
| Expert module | 5 fiches sur un même module |
| Modérateur solidaire | 10 modérations effectuées |
| Veilleur | 5 vérifications de fiches périmées |

Score = 10 pts/fiche + 3 pts/réponse forum + 1 pt/évaluation + 5 pts/signalement + bonus utilité.

## 6. Si la plateforme ne suffit pas

Renvoyez-vous toujours vers :
- **Assistance Op@le DAF A3** (Pléiade)
- **Bureau réglementation comptable** du rectorat
- **Communautés AC** : AJI, Intendance Zone

## 7. Support technique

Contact : administrateur applicatif académique (rôle `admin` dans l'application).
