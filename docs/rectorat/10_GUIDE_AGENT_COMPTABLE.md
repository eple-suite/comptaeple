# Guide de l'agent comptable

> Académie de la Guadeloupe — Plateforme **comptaeple**
> Document généré le 2026-04-25 — Version 1.0 (présentation rectorat)

## Premiers pas

1. Connexion + vérification e-mail
2. Rattachement à un ou plusieurs établissements (par admin)
3. Tour guidé du Cockpit (`Premier usage`)
4. Activation mode démo pour exploration sans risque

## Cycle annuel

- **Septembre** : passation SGEPLE, accréditation chef d'étab entrant, habilitations Op@le
- **Tout au long de l'année** : alimentation balance (import Op@le), traitement voyages, fonds sociaux, marchés, régies, SATD
- **Fin d'exercice** : compte financier (ordo + comptable), enquêtes
- **Continu** : capitalisation AIDE Op@le, entretiens BIATSS

## Points d'attention

- Toujours vérifier la **règle 8 €** avant SATD
- Anonymiser systématiquement les captures Op@le avant publication AIDE
- Consulter le **Rappel officiel** sur chaque fiche AIDE Op@le (redirection Pléiade en cas de doute)
