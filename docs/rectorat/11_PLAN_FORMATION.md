# Plan de formation académique

> Académie de la Guadeloupe — Plateforme **comptaeple**
> Document généré le 2026-04-25 — Version 1.0 (présentation rectorat)

## Public cible

- 100 % des AC de l'académie de la Guadeloupe
- Observateurs rectoraux désignés (DAF, DPD)
- Modérateurs AIDE Op@le (3 à 5 AC volontaires)

## Format

- Session 1 (2 h, distancielle) : Cockpit + Compte financier
- Session 2 (2 h, distancielle) : Voyages + Fonds sociaux + Marchés
- Session 3 (2 h, présentielle) : AIDE Op@le + Assistant Expert + Forum
- Atelier modérateurs (3 h, présentielle) : workflow modération + RGPD

## Supports

- Vidéos `docs/VIDEO_TOUR_*.md`
- Mode démo de la plateforme
- Cas pratiques fournis par le rectorat
