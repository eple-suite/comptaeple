# Visite guidée — Plateforme AIDE Op@le (script)

Durée cible : 4 minutes 30

## Séquence 1 — Positionnement (0:00 → 0:30)

- Plan large sur l'accueil `/ressources/opale`.
- Voix off : « Bienvenue sur la plateforme académique de capitalisation Op@le. Cet espace est complémentaire à l'assistance officielle DAF A3 et Pléiade. Il ne s'y substitue jamais. »
- Zoom sur le bandeau bleu de rappel.

## Séquence 2 — Bibliothèque (0:30 → 1:15)

- Cliquer sur « Bibliothèque de fiches ».
- Démontrer les filtres : module Op@le, tri par utilité.
- Ouvrir une fiche : montrer symptôme, cause, procédure étape par étape, badges (à jour, version Op@le, visibilité).
- Cliquer « Utile » → toast « Merci pour votre évaluation ».

## Séquence 3 — Recherche avancée (1:15 → 1:50)

- Aller sur « Recherche avancée ».
- Saisir un mot-clé (ex. « rejet PES »).
- Filtrer par module « Intégration PES » + version « V13.2 » + fraîcheur « À jour ».
- Mettre en évidence le résultat trié par utilité décroissante.

## Séquence 4 — Contribuer (1:50 → 3:00)

- Cliquer « Soumettre une fiche ».
- Parcourir les 7 étapes :
  1. Identification — saisir un titre fictif, choisir module Dépense.
  2. Contexte — décrire le symptôme.
  3. Diagnostic — cause + références.
  4. Procédure — ajouter 2 étapes numérotées.
  5. Versioning Op@le — saisir « V13.2 ».
  6. Visibilité — choisir « Académique ».
  7. Pré-publication — cocher l'auto-évaluation RGPD.
- Tester la détection : saisir « EPLE 9710001A » → la publication est bloquée.
- Soumettre pour modération.

## Séquence 5 — Modération (3:00 → 3:40)

- Connecté en tant que modérateur, ouvrir « Modération académique ».
- Cliquer « Approuver et publier ».
- Montrer le badge « Publiée » et la fiche apparaître dans la bibliothèque.

## Séquence 6 — Forum Q&R (3:40 → 4:10)

- Aller sur « Forum Q&R inter-AC ».
- Poser une question : « Erreur P-A1 sur intégration PES, comment retracer ? ».
- Montrer la liste des questions ouvertes.

## Séquence 7 — Tableau de bord (4:10 → 4:30)

- Aller sur « Tableau de bord académique ».
- Mettre en avant les KPI : fiches publiées, taux d'utilité, modules problématiques.
- Voix off de clôture : « Une plateforme par les AC, pour les AC, sous supervision rectorale. »
